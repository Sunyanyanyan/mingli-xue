第10课：流年怎么看——每年刮什么风

椰子要破壳 · 八字入门系列 · 第10课

上两课讲了大运（十年大环境）。今天讲流年（每年的天气）。

命局、大运、流年三件套

假设你是一辆车：

是什么管多久
命局出厂配置（轿车、卡车、跑车）一辈子
大运十年一段的路况（高速、山路、泥地）十年
流年每天的天气（下雨、大太阳）一年
命局是车，大运是路，流年是天气。

同样下雨，你在高速上和在泥地里，完全两码事。所以看流年必须是「流年+大运+命局」三合一，不能单看。

怎么读一个流年

第一步：流年本身

每年一对干支，拆开读。

  • 天干管表面 → 升职、换工作、公开吵架、领奖，别人看得到的
  • 地支管实质 → 心里想的、藏着的、不说的、底下发生的

举个虚构例子——甲木日主，流年丙午。

天干丙火：木生火 → 甲阳丙阳 → 同阴阳 → 食神。食神透天干 = 今年才华摆桌面上，谁都看得到。

地支午火：午藏丁己（火+土）。丁是伤官（阳生阴），己是正财。地支里伤官+正财搭着 → 底下在偷偷琢磨怎么把才华变成钱。

天干是台前，地支是幕后。台上食神笑咪咪，台下伤官在磨刀——这个人今年不简单。

第二步：流年碰大运

流年进来先碰大运，再碰命局。别跳步。

还是那个甲木日主，假设他大运在庚申（七杀+七杀）。

天干：流年丙火 → 克 → 大运庚金。食神制杀！

地支：流年午火 → 克 → 大运申金。同样的，底下也在制杀。

食神制杀，全面开花。

七杀是压力、是管束——今年他把压力按住了。不是被领导追着跑，是他主动把事儿干了、把嘴堵上了。

如果换一个人，同样丙午年，大运在壬子（偏印+偏印）呢？

天干：流年丙 → 被 → 大运壬克。食神被偏印摁住了——想说，有人不让说。

地支：子午冲！水火打架。底下翻江倒海，表面还装没事。

同一个丙午年，不同大运，天上地下。

四个组合，记住就够了

大运流年结果怎么办
🟢 好🟢 好起飞年全力冲
🟢 好🔴 差大风小雨别怕，过
🔴 差🟢 好喘口气趁好捞，别贪
🔴 差🔴 差最难的一年趴着，能过

第三步：流年碰命局——五个必看

这五个够了，不用全背。

① 天干合

流年天干跟你命局天干五合 → 被合的那个十神「动了」。

比如庚金日主，流年乙木 → 乙庚合。正财来合你 → 有人送钱、有人要你签合同、或者结婚。

合 = 有人找你。好事坏事看被合的那个十神是什么。

② 地支冲

六冲：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲。

冲 = 动。被冲的地支里藏的东西全翻出来。

比如命局带丑，流年未 → 丑未冲！丑里藏的财官印全出土。埋着的事今年翻出来见光。

③ 地支合

三合：申子辰合水、巳酉丑合金、亥卯未合木、寅午戌合火。

合 = 结盟。三个兄弟到齐了，那个五行的力量拉到满。

命局有巳酉，流年丑到了 → 巳酉丑合金！金的力量突然暴涨。如果金是你的七杀——压力骤增。如果是你的正印——贵人全到。

合局就像拼图，最后一块到了，画面完整了。

④ 伏吟与反吟

伏吟 = 流年干支跟命局某个干支一样。反吟 = 流年跟命局天克地冲。

伏吟 → 重复。那个位置的事今年再来一遍。
反吟 → 翻车。那个位置的东西直接掀翻重来。

伏吟是炒冷饭，反吟是掀桌子。一个是烦，一个是痛。

⑤ 长生 / 帝旺 / 入墓

流年地支是你日主的什么？

  • 长生 → 想重新开始，新年新气象
  • 帝旺 → 到顶峰了，该收了
  • 入墓 → 累了想休息，入库歇着

甲木长生在亥、乙木长生在午、丙戊长生在寅、丁己长生在酉、庚长生在巳、辛长生在子、壬长生在申、癸长生在卯。

查表太麻烦？记一个就行了：记住了你日主的长生地，以后每年一看地支就知道今年是「想冲」还是「想收」。

十神速查法

看到流年天干，不用查表。三秒定十神：

以你日主为基准，五行分五类：

  • 同五行 → 比劫（帮你的 / 争你的）
  • 你生的 → 食伤（你说的 / 你做的）
  • 你克的 → 财星（你得到的）
  • 克你的 → 官杀（管你的 / 压你的）
  • 生你的 → 印星（帮你的 / 护你的）

阴阳看正偏：

  • 同阴阳 → 偏（食神、偏财、七杀、偏印）
  • 异阴阳 → 正（伤官、正财、正官、正印）

流月叠加

流年是全年的天气预报，流月是每个月的具体天气。看流月跟看流年一样，多了一层叠加：

  • 流月天干地支本身
  • 碰流年（火上浇油 or 雪中送炭）
  • 碰大运
  • 碰命局
口诀：生比克冲合，优先级从高到低。
生你 > 跟你同类 > 克你 > 冲你 > 合你。
