第2课：天干——十个性格鲜明的小伙伴

椰子要破壳 · 八字入门系列 · 第2课

上次讲了一个事——八字是八个字的出厂说明书。今天拆第一个零件：天干。就是那十个你小时候背过的字：甲乙丙丁戊己庚辛壬癸。

小时候觉得是序号。但在八字里，它们不是序号——是十种不同的能量，十个性格鲜明的人。

带过娃的都懂——同样是哭：

· 有的宝宝雷声大雨点小，哭完就忘（甲木）

· 有的小声哼唧，没人理才嗷一声（乙木）

· 有的咧嘴就嚎、嚎完就笑，翻篇比翻书快（丙火）

十种天干，就是十种不同的"哭法"。

分两组：阳干和阴干

十天干跟五行配对，每行分阴阳：

木 甲（阳·大哥） / 乙（阴·二姐）

火 丙（阳·大哥） / 丁（阴·二姐）

土 戊（阳·大哥） / 己（阴·二姐）

金 庚（阳·大哥） / 辛（阴·二姐）

水 壬（阳·大哥） / 癸（阴·二姐）

规律：甲丙戊庚壬是阳，乙丁己辛癸是阴。
阳刚，阴柔。

问： 怎么记住谁阳谁阴？

看单双。甲乙丙丁戊己庚辛壬癸——第1、3、5、7、9位是阳，第2、4、6、8、10位是阴。扳手指头就行。

十个天干的人格画像

用画面记，不用背。

甲木（阳木） 参天大树——正直担当，天塌下来我顶着。但太硬，弯不下腰。

乙木（阴木） 藤蔓花草——柔韧善变，很会调节气氛。但容易太迁就别人，把自己磨没了。

丙火（阳火） 太阳——热情开朗，走到哪都自带暖炉。但掏心掏肺对别人好，没回应就难受。

丁火（阴火） 灯烛——细腻敏感，安静专注有灵气。但风一吹就灭，容易玻璃心。

戊土（阳土） 城墙——厚重靠谱，"没事有我"。但消化慢，脾胃容易出问题。

己土（阴土） 田园——包容滋养，种啥长啥。但容易被当工具人。

庚金（阳金） 刀剑——刚毅果决，利索干脆。但太直，容易伤到人。

辛金（阴金） 珠宝——精致讲究，打磨过的金子。但想太多爱钻牛角尖。

壬水（阳水） 江河——大气自由，开阔不纠结。但太粗心，不管细节。

癸水（阴水） 雨露——智慧细腻，表面平静底下暗流。聪明但思虑过重。

问： 怎么看自己是什么干？

拿到八字找日柱——日柱上面那个字，就是你的天干。这个字决定了你的基本气质。别的天干是别人，日干是你。

本课小结

  • 十天干 = 十种人格，不是序号
  • 分阳干（单数位，像大哥）和阴干（双数位，像二姐）
  • 看自己是什么干：八字日柱上面那个字
  • 天干决定你的基本气质
下次别人说「甲木人都有领导力」，你可以问：
那他八字里其他七个字在看戏吗？
一个天干只是开个头。真正有意思的在后面。
