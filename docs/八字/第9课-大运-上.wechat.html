第9课（上）：大运——十年一个命

椰子要破壳 · 八字入门系列 · 第9课

大字报先贴：

命局是出厂配置。大运就是你每十年换一条路。

命局好 + 大运差 = 豪车开泥地
命局差 + 大运好 = 破车跑高速

命局改不了。大运能看、能用。不看大运只看命局，就等于开了一辈子车不知道自己在高速上还是泥地里。

大运从哪来

从月柱来。

月柱是你的出生月份。大运就是从月柱开始，按规则往外排，每十年一换，一路走到老。

月柱 = 大运的起点。因为月柱代表了你来到这个世界时的季节温度——你这颗种子是春天撒的还是秋天撒的。大运就是未来的季节往哪边走。

怎么排大运

第一步：定顺逆

年干性别方向
阳（甲丙戊庚壬）男顺排
阴（乙丁己辛癸）男逆排
阳女逆排
阴女顺排
口诀：阳男阴女顺，阴男阳女逆。

顺排 = 从月柱往后数（甲→乙→丙→丁→戊→己→庚→辛→壬→癸）
逆排 = 从月柱往前数（甲→癸→壬→辛→庚→己→戊→丁→丙→乙）

第二步：排干支

假设一个人月柱是丙午，他是阳男→顺排。

步大运
第一步丁未
第二步戊申
第三步己酉

每步十年。三步就是三十年。排八步够用了。

逆排反过来。月柱丙午，逆排的话：

步大运
第一步乙巳
第二步甲辰
第三步癸卯

第三步：起运年龄——几岁开始走大运

这是问得最多、也最容易被算错的。

规则：从出生日数到下一个（或上一个）换月节，3天 = 1岁起运。

换月节就是每个月的那个「节」：立春、惊蛰、清明、立夏、芒种、小暑、立秋、白露、寒露、立冬、大雪、小寒。

顺排 → 数到下一个换月节。逆排 → 数到上一个换月节。

举个例子： 1995年10月1日出生，女，乙年（阴）→ 阴女顺排。

顺排 = 数到下一个换月节。10月1日的下一个节是寒露（10月8-9日）。

10月1日到10月9日 ≈ 8天。8 ÷ 3 ≈ 2.67年 = 2岁8个月起运。

也就是说这个人第一轮大运从2岁8个月开始，走到12岁8个月。第二轮12岁8个月到22岁8个月……

换算口诀：3天=1岁，1天=4月，1时辰(2h)=10天。

为什么有的孩子早运有的晚运

起运早的（0岁、1岁就起运）→ 出生离换月节近，差的天气。

起运晚的（5岁、7岁才起运）→ 出生离换月节远。

起运早 → 从小就明显。起运晚 → 童年比较「混」，大运来了才显出特点。

跟聪明、命好没关系。就是出生时间跟节气的远近问题。
