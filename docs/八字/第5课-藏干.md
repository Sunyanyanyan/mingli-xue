# 八字｜藏干——看不到的，才是真相

> 椰子要破壳 · 八字入门系列 · 第5课

五行生克学会了。但你有没有遇到过这种人——表面斯斯文文什么都不争，骨子里倔得要命？

因为地支看着只是一个字，**里面却藏着好几个天干**。就像一个人站在面前是一副脸，心里却装着好几个念头。要真正看懂一个人，就得看那些藏起来的东西。

---

## 藏干是什么？

藏干 = 地支里藏着的天干。

打个比方：
- **地支** = 一栋房子
- **藏干** = 房子里的人

看房子外观，"哦，白墙灰瓦"。推门进去，里面住着三口人，每个人性格不一样。

**每个地支代表一个节气阶段。每个阶段里：**
- 还残留着**上一个季节的气**（叫余气）
- 正处着**当下最旺的气**（叫本气）
- 孕育着**下一个季节的种子**（叫中气）

### 为什么有的藏1个、有的藏3个？

十二地支按季节分三组，每一组角色不同。

先把十二地支排进四季：

```
春（木）：寅  卯  辰
        正月 二月 三月

夏（火）：巳  午  未
        四月 五月 六月

秋（金）：申  酉  戌
        七月 八月 九月

冬（水）：亥  子  丑
        十月 冬月 腊月
```

每季三个月，各扮演不同角色：

| 位置 | 叫什么 | 藏几个 | 像什么 |
|:--|:--|:--|:--|
| 四正（子午卯酉） | **帝旺** | 1-2个 | 四季最中间，气最纯——站山顶只能站你一个 |
| 四长生（寅申巳亥） | **长生** | 2-3个 | 刚换季，新旧交替——带着上一季残余+本季+下一个 |
| 四墓库（辰戌丑未） | **墓库** | 3个 | 季末收割，仓库收尾——土为主夹着残渣 |

**不用背——想季节就懂了：**
- 子午卯酉 = 四季正中间，最旺最纯 → 藏得少（1-2个）
- 寅申巳亥 = 刚换季第一个月，新旧混杂 → 藏2-3个
- 辰戌丑未 = 季末收尾的土，什么东西都往里塞 → 藏3个

## 十二地支藏干

| 地支 | 藏干 | 说明 |
|:----:|:----|:-----|
| 子 | 癸 | 纯水，只藏一个 |
| 丑 | 己、癸、辛 | 土+水+金 |
| 寅 | 甲、丙、戊 | 木+火+土 |
| 卯 | 乙 | 纯木，只藏一个 |
| 辰 | 乙、戊、癸 | 木+土+水 |
| 巳 | 丙、戊、庚 | 火+土+金 |
| 午 | 丁、己 | 火+土 |
| 未 | 己、丁、乙 | 土+火+木 |
| 申 | 庚、壬、戊 | 金+水+土 |
| 酉 | 辛 | 纯金，只藏一个 |
| 戌 | 辛、丁、戊 | 金+火+土 |
| 亥 | 壬、甲 | 水+木 |

> **问：** 必须背下来吗？
>
> 不用死背。
>
> 四个规律帮你记：
>
> **① 四正（子午卯酉）最单纯** — 大多藏一个（午例外藏两个），气最纯
> **② 四长生（寅申巳亥）最有活力** — 大多藏三个（亥藏两个），新旧混杂
> **③ 四墓库（辰戌丑未）最复杂** — 全部藏三个，仓库收尾什么都有
> **④ 本气最重** — 每个地支的本气（最旺那个）跟它自己同五行（子本气癸水、丑本气己土）
>
> 规律帮你理解规律，**具体藏什么看上面那个表。** 用的时候查表就行，用多了自然记住。

## 举个例子

假设一个人的八字地支是：**寅 午 辰 申**

**寅** → 藏甲丙戊
- 甲木 → 有根，不是无根之草
- 丙火 → 有表现力，好表达
- 戊土 → 有掌控资源的欲望
- → 表面稳重，内心正在燃烧

**午** → 藏丁己
- 丁火 → 有细腻的创造力
- 己土 → 能把想法稳住落地
- → 既有创意，又懂得落地

**辰**（日支，婚姻/内心）→ 藏乙戊癸
- 乙木 → 常有人跟你较劲，但也因此变强
- 戊土 → 骨子里大方，不抠门
- 癸水 → 心里有温柔的一面
- → 三个人格共存：好胜、大方、心软

## 藏干告诉我们什么？

学会看藏干，就不会被表面骗了。

- 一个人天干全是丙火（热情），地支藏了一堆癸水（阴冷）
→ **表面大大咧咧，心里特别敏感**
- 一个人天干是甲木（直爽大气），地支藏了辛金（小刀）
→ **外面大气，内心挑剔**

**人不是单面的。每个人都有不被看见的那一面。**

---

## 本课小结

- 藏干 = 地支里藏着的天干
- 一个地支通常藏2-3个，分**本气/中气/余气**
- 看人不能只看表面，要看看藏干
- 四正最单纯、四长生最有活力、四墓库最丰富
