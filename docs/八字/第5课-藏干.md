# 第5课：藏干——看不到的，才是真相

> 椰子要破壳 · 八字入门系列 · 第5课

---

上节课学了五行生克，知道谁在帮你、谁在压你。

但有一个问题：地支看着只是一个字，难道真就这么单纯？

当然不。**每个地支里面，藏着看不见的天干。**

---

## 藏干是什么？

藏干 = 地支里藏着的天干。

打个比方：
- **地支** = 一栋房子
- **藏干** = 房子里的人

看房子外观，"哦，白墙灰瓦"。推门进去，里面住着三口人，每个人性格不一样。

**每个地支代表一个节气阶段。每个阶段里：**
- 还残留着**上一个季节的气**（叫余气）
- 正处着**当下最旺的气**（叫本气）
- 孕育着**下一个季节的种子**（叫中气）

---

## 十二地支藏干

| 地支 | 藏干 | 说明 |
|:----:|:----|:-----|
| 子 | 癸 | 纯水，只藏一个 |
| 丑 | 己、癸、辛 | 土+水+金 |
| 寅 | 甲、丙、戊 | 木+火+土 |
| 卯 | 乙 | 纯木，只藏一个 |
| 辰 | 乙、戊、癸 | 木+土+水 |
| 巳 | 丙、戊、庚 | 火+土+金 |
| 午 | 丁、己 | 火+土 |
| 未 | 己、丁、乙 | 土+火+木 |
| 申 | 庚、壬、戊 | 金+水+土 |
| 酉 | 辛 | 纯金，只藏一个 |
| 戌 | 辛、丁、戊 | 金+火+土 |
| 亥 | 壬、甲 | 水+木 |


> 
> **问：** 必须背下来吗？**
>
> 不用死背。用多了就记住了。刚开始截图存手机，用的时候查。就像刚开始当妈，也记不住宝宝每顿吃多少毫升，喂多了就知道了。
>
> 四个规律帮你记：
>
> **① 四正（子午卯酉）最单纯** — 各只藏一个，像四季中间
> **② 四长生（寅申巳亥）最有活力** — 都藏三个，像种子刚发芽
> **③ 四墓库（辰戌丑未）最复杂** — 都藏三个，像仓库装着各种东西
> **④ 阴阳不乱** — 阳支藏阳干，阴支藏阴干

---

## 举个例子

假设一个人的八字地支是：**亥 酉 丑 丑**

**亥** → 藏壬甲
- 壬水（生他）暗中有贵人支持
- 甲木（同他）有根，不是无根之草
- → 看起来好说话，骨子里有主意

**酉** → 藏辛
- 纯金一个
- 七杀的根，一直在暗中"修剪"他
- → 骨子里对自己要求很高

**丑**（日支，婚姻/内心）→ 藏己癸辛
- 己土（偏财）→ 务实过日子
- 癸水（偏印）→ 敏感多思靠直觉
- 辛金（七杀）→ 有股倔劲
- → 表面过日子，心里多思多虑，骨子里不吐不快

---

## 藏干告诉我们什么？

学会看藏干，就不会被表面骗了。

- 一个人天干全是丙火（热情），地支藏了一堆癸水（阴冷）
→ **表面大大咧咧，心里特别敏感**
- 一个人天干是乙木（温柔随和），地支藏了庚金（刀剑）
→ **外面好好先生，内心很有原则**

**人不是单面的。每个人都有不被看见的那一面。**

---

## 本课小结

- 藏干 = 地支里藏着的天干
- 一个地支通常藏2-3个，分**本气/中气/余气**
- 看人不能只看表面，要看看藏干
- 四正最单纯、四长生最有活力、四墓库最丰富
