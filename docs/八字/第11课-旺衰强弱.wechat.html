第11课：旺衰强弱——你电池有多大

椰子要破壳 · 八字入门系列 · 第11课

上两课学了大运和流年。但有一个最关键的问题还没回答：同样一个正官，为什么压在甲木身上让它垮了，挡在丙火面前反而让它更强？

因为两个人的电量不一样。

先看地图

旺衰强弱 = 判断日主是「满电」还是「快没电了」。这东西不判，后面用神忌神全乱套。

手机电池打比方

身旺 → 手机电量80%。收到十条消息，回完还剩60%。扛得住。

身弱 → 手机电量20%。收到三条消息就得省着回。再开个导航直接关机。

八字里的十神，就看你这块电池能不能扛住。电池大 → 七杀是磨刀石。电池小 → 七杀是压路机。

怎么判断：看三样

① 得令——你生在哪个季节

日主五行跟月支五行一样 → 得令，先天电量就大。

日主得令月
木（甲乙）寅、卯（春）
火（丙丁）巳、午（夏）
金（庚辛）申、酉（秋）
水（壬癸）亥、子（冬）
土（戊己）辰戌丑未（四季末）

② 得地——地支里有多少队友

日主在地支里有没有同类。藏干也算。

③ 得助——天干上有没有帮手

天干上有没有跟日主同五行（比肩劫财）或生它的（正印偏印）。

三样凑起来 → 两样以上得 = 身旺。
一样或都没有 = 身弱。

举个例子：丁火日主

假设一个人日主丁火，月支亥（水）。

  • 得令？亥月水旺，不是火 → 不得令 ❌
  • 得地？地支有卯（藏乙木生火）✅ + 其他不帮 → 得地弱
  • 得助？天干有甲木（正印生火）→ 有得助 ✅

三样得一样 → 身弱。

这个丁火身弱的人：电池小。碰上克火的水（正官七杀）容易崩。需要木来生火（印星续命）、火来帮火（比劫分担）。

身旺 vs 身弱，不是谁好谁坏

身旺身弱
好处扛造，能打持久战敏感，察觉别人察觉不到的
注意太自我，听不进话易累，容易被压垮
策略泄一泄（食伤）补一补（印比）

不是「身旺好身弱不好」，是两套不同的出厂设置。身弱的人往往更细腻、更有直觉——因为电池小，信号接收就敏感。

本课小结

  • 旺衰 = 看得令、得地、得助三样
  • 两样以上就是身旺，一样或没有就是身弱
  • 身弱多补印比，身旺多补食伤
  • 丁火亥月 → 身弱。需要木（印）续命，火（比劫）分担
