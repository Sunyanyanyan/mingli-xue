## Main Topic
十神（正官、七杀、正印、偏印、伤官、食神、正财、偏财、劫财、比肩）是八字中天干与天干之间的十种关系。以乙木日主为中心，五种五行关系（克我、生我、我生、我克、同我）各分阴阳，共十种。

## Learning Objectives
After viewing this infographic, the viewer should understand:
1. 十神的来源：五行生克 + 阴阳异同
2. 以乙木日主为例，五种关系对应的十神名称
3. 每种十神的阴阳区别（正/偏、官/杀、印/枭等）

## Target Audience
- **Knowledge Level**: Beginner (学过前5课基础的命理爱好者)
- **Context**: 学习十神时容易混淆，需要一张图理清关系
- **Expectations**: 一目了然看到十神全貌

## Content Type Analysis
- **Data Structure**: 中心辐射（hub-spoke）—— 日主居中，五种关系向外辐射，每种再分阴阳
- **Key Relationships**: 五行生克决定了五种关系类型，阴阳决定了每种关系内的正/偏之分
- **Visual Opportunities**: 中心→外圈辐射结构适合信息图；用五行颜色区分五大类；阴阳用左右对称展示

## Key Data Points (Verbatim)
- 克我者为官杀：正官（庚）、七杀（辛）
- 生我者为印：正印（壬）、偏印（癸）
- 我生者为食伤：伤官（丙）、食神（丁）
- 我克者为财：正财（戊）、偏财（己）
- 同我者为比劫：劫财（甲）、比肩（乙）
