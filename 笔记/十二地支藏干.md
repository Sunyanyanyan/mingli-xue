# 十二地支藏干速查

## 三组框架

四正（季中，最纯）→ 藏1-2个
四长生（季初，新旧交替）→ 藏3个  
四墓库（季末，仓库收尾）→ 藏3个

## 速查表

### 四正
| 地支 | 藏干 | 五行 |
|:--|:--|:--|
| 子 | 癸 | 水 |
| 午 | 丁、己 | 火土 |
| 卯 | 乙 | 木 |
| 酉 | 辛 | 金 |

### 四长生
| 地支 | 藏干 | 口诀 |
|:--|:--|:--|
| 寅 | 甲、丙、戊 | 木火土 |
| 申 | 庚、壬、戊 | 金水土 |
| 巳 | 丙、戊、庚 | 火土金 |
| 亥 | 壬、甲 | 水木（少一个） |

### 四墓库
| 墓库 | 藏干 | 口诀 |
|:--|:--|:--|
| 辰（阳土） | 戊、乙、癸 | 土木水 |
| 未（阴土） | 己、丁、乙 | 土火木 |
| 戌（阳土） | 戊、辛、丁 | 土金火 |
| 丑（阴土） | 己、癸、辛 | 土水金 |

## 记忆法

- **季初混杂**（寅申巳亥）→ 藏3个
- **季中最纯**（子午卯酉）→ 藏1-2个  
- **季末仓库**（辰戌丑未）→ 藏3个，全是土为主

墓库互补：辰-戌一对，丑-未一对
