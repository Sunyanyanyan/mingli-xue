第13课：格局入门——八字的主线剧情

椰子要破壳 · 八字入门系列 · 第13课

用神忌神讲单个五行对你好不好。今天讲格局——整个盘在演什么戏。

先看地图

格局 = 从月令出发，看全局能量往哪个方向走。月令是整张盘的导演。

格局怎么定

两步：

  • 看月令 → 月支是什么，格局方向就在哪
  • 看透出 → 月支藏干在天干露了没？

透出了 → 按透出的定格局

没透 → 取其他透出的最强十神

举个例子：己土日主

假设己土日主，辰月（土当令）。辰藏乙戊癸。天干透戊土劫财、庚金伤官。

戊土是辰的本气，天干透了 → 月令透出 → 劫财格。

劫财格的人：主线是「跟人打交道」。朋友多、竞争也多。

如果天干没透戊土 → 看庚金伤官 → 伤官格。主线变「才华突围」。

常见格局

正官格 规矩内做事 → 体制、管理

七杀格 压力逆袭 → 创业、突破

食神格 才华积累 → 创作、手艺

伤官格 表达突围 → 设计、演讲

正印格 学识深耕 → 教育、研究

正财格 资源经营 → 稳定收入

格局 + 旺衰

食神格+身弱 → 才华有方向，体力跟不上。慢慢烧。

正官格+身旺 → 有人管你，你也扛得住。天作之合。

七杀格+身弱 → 压力主线，电池又小。必须有印或食伤化解。

本课小结

  • 格局 = 从月令看主线剧情
  • 月令藏干天干透出 → 以透出者定格
  • 格定了看旺衰：扛得住才能演好主角
