# 第0课：易经到底在说什么

> 椰子陪你学 · 易经入门系列

---

## 八个字讲完易经

**万事万物怎么变化。**

太阳升起来 → 落下去。
春天来了 → 夏天 → 秋天 → 冬天。
你生了娃 → 娃长大 → 你也变了。

**一切都在变。易经就是研究这个的。**

---

## 你已经在用了

八字里的阴阳五行，根全在易经里。

- 阴阳 → 易经说「一阴一阳之谓道」
- 五行生克 → 易经的变卦规律
- 气场流动 → 易经的核心思想

学易经不是学新东西，是去挖八字的根。

---

## 易经记的是什么？

三样东西：

1. **卦** —— 64个符号，每个代表一种状态
2. **爻** —— 每条卦里的6根线，代表变化的细节
3. **辞** —— 对每个卦和爻的解释

打个比方：
> 卦 = 天气图标（☀️🌧️⛅）
> 爻 = 每小时的温度
> 辞 = 天气预报员的解说

---

## 第一个概念：阴阳

易经所有的变化，只有两个零件：

```
—— 阳（一条长线）
— — 阴（两条短线）
```

**所有的卦，都是这两种线叠出来的。**

跟电脑只有0和1一个道理——两个符号，能组合出整个世界。

---

## 64卦 = 64种人生状态

每个人生阶段，都能对应一个卦：

- 刚结婚 → 咸卦（感应）
- 遇到困难 → 蹇卦（艰难）
- 觉得卡住了 → 解卦（解脱）
- 一切顺利 → 泰卦（通顺）

**不是卦在决定你，是卦在描述你。**

---

## 学习路线

```
第1层：阴阳 → 万物的开关
第2层：八卦 → 8种基本状态
第3层：64卦 → 64种剧本
第4层：爻 → 每场戏的细节
第5层：系辞 → 古人的批注
```

后面一课一层，跟学八字一样剥洋葱。
