# 第2课：八卦——八种基本状态

> 椰子陪你学 · 易经入门系列

---

上节课说了，三根阴阳线一组合，就有八种结果。

这八种结果，就是八卦。**八个符号，代表了宇宙间八种最基本的能量状态。**

---

## 怎么来的？

拿三根阴阳线往上叠：

```
阳阳阳 = ☰ 乾（天）
阳阳阴 = ☱ 兑（泽）
阳阴阳 = ☲ 离（火）
阳阴阴 = ☳ 震（雷）
阴阳阳 = ☴ 巽（风）
阴阳阴 = ☵ 坎（水）
阴阴阳 = ☶ 艮（山）
阴阴阴 = ☷ 坤（地）
```

背不住没关系，先看懂规律：

- **阳多 = 主动、刚健、向外**
- **阴多 = 被动、柔顺、向内**

---

## 八个卦，八种人

把它们翻译成人，特别好记：

| 卦 | 符号 | 像什么 | 对应的人 |
|----|------|--------|----------|
| 乾 | ☰ | 天 | 爸爸、领导、创业者 |
| 坤 | ☷ | 地 | 妈妈、包容者、后勤 |
| 震 | ☳ | 雷 | 大儿子、行动派 |
| 巽 | ☴ | 风 | 大女儿、渗透型人才 |
| 坎 | ☵ | 水 | 二儿子、冒险家 |
| 离 | ☲ | 火 | 二女儿、传播者 |
| 艮 | ☶ | 山 | 小儿子、坚守者 |
| 兑 | ☱ | 泽 | 小女儿、沟通者 |

**对应到你生活里：**

- 你老公要是那种一声不吭扛事的人 → 乾卦（天）
- 你是那种默默操心全家的人 → 坤卦（地）
- 你娃要是整天跑来跑去不停 → 震卦（雷）
- 邻居那个话多到处聊的 → 兑卦（泽）

**不是你这个人等于一个卦，是你某个阶段、某件事上像哪个卦。**

---

## 八卦跟带娃的关系

| 带娃场景 | 对应卦 | 解释 |
|---------|--------|------|
| 娃半夜哭闹不睡 | 震 ☳ | 突然爆发、动静大 |
| 娃自己玩得专心 | 艮 ☶ | 停下来不动了 |
| 你做饭收拾屋子 | 坤 ☷ | 默默承载一切 |
| 老公加班不回家 | 乾 ☰ | 在外面刚健运行 |
| 带娃去公园撒欢 | 兑 ☱ | 开心、释放、交流 |
| 娃发烧你焦虑 | 离 ☲ | 火在烧、向外散 |
| 娃认生躲你身后 | 坎 ☵ | 危险感、向内收 |
| 阿姨来帮忙带娃 | 巽 ☴ | 慢慢渗透进来 |

**没有哪个卦好哪个坏。每个卦都有它的用法。**

## 八卦的口诀（古人传下来的）

```
乾三连    坤六断
震仰盂    艮覆碗
离中虚    坎中满
兑上缺    巽下断
```

什么意思？

- 乾三连 = ☰ 三条线连着的
- 坤六断 = ☷ 六条短线（三个两段）
- 震仰盂 = ☳ 像一个开口朝上的碗
- 艮覆碗 = ☶ 像一个扣着的碗
- 离中虚 = ☲ 中间是空的
- 坎中满 = ☵ 中间是实的
- 兑上缺 = ☱ 上面缺一口
- 巽下断 = ☴ 下面断开了

**不用背，看多了自然记住。**

---

## 本课小结

- 八卦 = 三根阴阳线全排列 = 8种
- 每个卦代表一种基本能量状态
- 没有好坏，只有不同场景的适用
- 口诀是帮你记符号的，不用硬背
