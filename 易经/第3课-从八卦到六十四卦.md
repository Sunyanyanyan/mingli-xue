# 第3课：从八卦到六十四卦

> 椰子陪你学 · 易经入门系列

---

上节课讲了八个卦。这节课把它们叠起来——**两个八卦上下组合，就是六十四卦。**

每卦代表你人生的一种状态。

---

## 怎么叠的？

八卦有8个：乾兑离震巽坎艮坤。

每个卦做上卦，每个卦做下卦，全部组合：

```
8 × 8 = 64
```

打个比方：

> **菜谱。**
>
> 你有8种食材：鸡、鱼、肉、蛋、菜、豆、米、面。
> 两种食材搭配，能出多少道菜？
> 8 × 8 = 64道。
>
> 64卦就是64道人生菜谱。

---

## 六十四卦长什么样

每卦六根线（六爻），上三根叫**上卦**，下三根叫**下卦**。

```
上卦 ☰ 乾（天）
下卦 ☷ 坤（地）
→ 组成 ䷋ 否卦（天地否）
```

```
上卦 ☵ 坎（水）
下卦 ☲ 离（火）
→ 组成 ䷾ 既济卦（水火既济）
```

**不是随便拼的。上下两卦的关系，就是你这个状态的核心矛盾。**

---

## 怎么读一个卦？

每卦有六根爻，从下往上数：

```
  第五爻 ─────
  第四爻 ─ ─ ─
  第三爻 ─────
  第二爻 ─ ─ ─
  第一爻 ─────   ← 从下往上读
```

**最下面第一爻 = 事情的开始**
**最上面第六爻 = 事情的结局**

---

## 举个你熟悉的卦：䷂ 屯卦

上卦 ☵ 坎（水 = 险）
下卦 ☳ 震（雷 = 动）

**组合的意思：雷雨交加，万物初生。**

对应你的人生场景：

| 爻 | 意思 | 你的版本 |
|----|------|----------|
| 初爻 | 刚起步，难 | 刚生完娃，手忙脚乱 |
| 二爻 | 有人帮你 | 老公/月嫂搭把手 |
| 三爻 | 又卡住了 | 娃肠绞痛，整夜不睡 |
| 四爻 | 慢慢顺了 | 找到规律了 |
| 五爻 | 小有成果 | 娃能睡整觉了 |
| 上爻 | 这个阶段过了 | 娃会走路了，进入下一阶段 |

**一个卦，就是一件事从开始到结束的完整过程。**

---

## 你不需要背64个卦

很多人一听64卦，吓住了。

**不用背。你只需要记住一个规律：**

> **上卦 = 外部环境**
> **下卦 = 你自己**
> **上下关系 = 你跟环境的互动**

看到一个卦，先看上卦像什么（天/地/水/火/风/雷/山/泽），再看下卦像什么，然后想这两个东西放一起会怎样。

比如：
- 上天下地 → 人在中间，天地不通 → ䷋ 否卦（不顺）
- 上火下水 → 火往上烧，水往下流，不相干 → ䷾ 既济卦（各归其位）

---

## 本课小结

- 64卦 = 8个上卦 × 8个下卦
- 每卦六根爻，从下往上读
- 上卦代表外部环境，下卦代表你自己
- 不用背，理解两个卦的关系就行
