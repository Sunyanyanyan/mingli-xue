# 十神到底是什么？一张图看懂

> 椰子要破壳 · 八字入门

---

学八字最怕什么？

**十神。**

正官、七杀、正印、偏印、伤官、食神、正财、偏财、劫财、比肩——十个名字，看着就想放弃。

但其实，十神不新。

你只要会两样东西：**五行生克**和**分阴阳**。两个拼一起，十神就出来了。

---

## 一张图看全貌

```
谁在克你？
 ├─ 阴阳相反 → 正官（温和管你）
 └─ 阴阳相同 → 七杀（用力压你）

谁在生你？
 ├─ 阴阳相反 → 正印（光明正大帮你）
 └─ 阴阳相同 → 偏印（暗地里助你）

你生了什么？
 ├─ 阴阳相反 → 伤官（才华喷出来）
 └─ 阴阳相同 → 食神（才华慢慢流）

你克了什么？
 ├─ 阴阳相反 → 正财（稳定收入）
 └─ 阴阳相同 → 偏财（意外之财）

谁跟你一样？
 ├─ 阴阳相反 → 劫财（竞品队友）
 └─ 阴阳相同 → 比肩（灵魂知己）
```

五个关系，每个分阴阳，十个。

---

## 拿一个人演一遍

假设一个人日主是丁火——小火苗，蜡烛那种。

**谁克丁火？** 水。壬水（阳）跟丁火（阴）相反 → 正官。癸水（阴）跟丁火相同 → 七杀。

正官像毛毛雨，让你冷静别烧过头。七杀像一盆水泼过来，直接压你。

**谁生丁火？** 木。甲木（阳）跟丁火相反 → 正印。乙木（阴）跟丁火相同 → 偏印。

正印象大块木头慢慢烧，持久温暖。偏印象枯草一点就着，灵感来得快去得快。

看见没？**生克定关系，阴阳定名字。** 不用背。

---

## 换成画面记一辈子

**管你的：**

- 正官 → 毛毛雨降温。领导说"方向对的，细化一下"。
- 七杀 → 一盆水泼过来。老板拍桌子"全推翻重来"。

**帮你的：**

- 正印 → 大木慢慢烧。有人默默把饭做好，不用开口。
- 偏印 → 枯草一点着。半夜突然想通bug，灵感一闪。

**你的才华：**

- 伤官 → 喷泉涌出。憋不住，非要说，还说得漂亮。
- 食神 → 清晨露珠。不大、不抢眼，但在光底下闪亮。

**你的钱：**

- 正财 → 院子里种的树。月月固定存一笔，稳。
- 偏财 → 野外摘的野果。朋友转你一笔钱，意外的甜。

**你的朋友：**

- 劫财 → 撞款的同行。较劲，但也因较劲都进步了。
- 比肩 → 同款耳环。不用解释，天生一对。

---

## 记不住怎么办？

拿你自己的日主，问三句：

1. 这人的五行跟我什么关系？（生我？克我？……）
2. 阴阳跟我一样吗？
3. **得出十神。**

就把五行生克的画面，翻译成人的关系——斧头砍树变成"有人管你"，浇水养花变成"有人帮你"。

**十神不是背的，是用画面想出来的。**
