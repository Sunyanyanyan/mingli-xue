很多人学八字，上来就背天干地支、背五行生克、背十神表。

错了。

**你应该先站远一步，看整张地图。** 不然学了三个月，还是不知道这些东西拼起来到底是什么。

---

## 八字不是算命，是读说明书

你买了一台车，车上有**说明书**。

八字就是你的生命说明书——不是告诉你「几岁发财、几岁离婚」的算命书，而是告诉你：

> 你是辆什么车、烧什么油、走什么路最对味。

- 有人是越野车，非要去跑赛道 → 累垮
- 有人是跑车，非要去爬山路 → 底盘刮烂
- 有人是面包车，非要跟跑车比速度 → 越比越难受

**八字不是改你的命，是让你别用错自己。**

---

## 八个字 = 四柱 × 2

八字就是八个字。大框架只有一句话：

```
      年   月   日   时  
      ↓   ↓   ↓   ↓  
    ┌───┬───┬───┬───┐
天干│ 甲 │ 丙 │ 乙 │ 丁 │  ← 外面看得到的（人格画像）
    ├───┼───┼───┼───┤
地支│ 寅 │ 午 │ 丑 │ 亥 │  ← 骨子里的底色（原生环境）
    └───┴───┴───┴───┘
```

四个柱子，每个柱子上下两个字：
- **上面的叫天干** —— 你表现出来的样子，像天气，外面看得见
- **下面的叫地支** —— 你骨子里的底色，像地基，别人看不见

**注意：** 日柱（第三柱）上面的天干叫**「日主」**——那是你。整个八字围绕你展开。

上面那个例子，日主是「乙」。你是一棵花草。

---

## 整张地图：从外往里看

八字不是平行世界的八件事。是**一层一层剥开的**：

```
第1层：天干（表面的你）
  ↓
第2层：地支（骨子里的你）
  ↓
第3层：藏干（你藏着的那面）
  ↓
第4层：五行生克（这些力量怎么互动）
  ↓
第5层：十神（帮你的人是谁，压你的人是谁）
```

**你学八字的顺序，就是剥洋葱的顺序。**

后面的课，每一课剥一层：
- 第1课 → 看懂这张表长什么样
- 第2课 → 剥第1层：天干
- 第3课 → 剥第2层：地支
- 第4课 → 讲力量怎么流动：五行
- 第5课 → 剥第3层：藏干
- 第6、7课 → 剥第4、5层：谁帮你、谁压你（十神）
- 第8课 → 拼起来：十神组合看性格

---

## 三个最重要的关系，就三个

以后你会学一堆东西——但全部归结为三个关系：

```
生你 → 帮你的人（印）
克你 → 管你的人（官、杀）
你生 → 你表达的东西（食、伤）
你克 → 你能驾驭的资源（财）
同你 → 跟你一样的人（比、劫）
```

五个关系，核心就三句话：

> **谁在帮我？谁在压我？我拿什么去应对？**

后面八课，全是围绕这三句话展开的。

---

## 别被术语吓到

「正官」「七杀」「食神」「伤官」——这些名字听着唬人。

**不用怕。它们全部来自一个公式：**

```
五行生克 + 阴阳 = 十神
```

你已经在第4课学过五行了，学十神的时候就是把五行的画面翻成人的关系——斧头砍树（金克木）变成「有人管你」，浇水养花（水生木）变成「有人帮你」。

**新东西？没有。每一步都是上一步的延伸。**

---

## 这张地图有什么用？

以后你任何时候觉得乱，回来看这张地图：

1. 八字 = 说明书，不是算命
2. 八个字 = 四柱 × 2
3. 五层洋葱：天干 → 地支 → 藏干 → 五行 → 十神
4. 三个核心问题：谁帮我？谁压我？我靠什么应对？
