上节课学了天干——十个性格鲜明的小伙伴。

今天讲下半部分：**地支**。

---

## 地支是什么？

十二地支：**子丑寅卯辰巳午未申酉戌亥**

如果说天干是你**表现出来的样子**，地支就是你**骨子里的底色**。

打个比方：
- 天干 = 你在朋友圈发的照片（精心选过的）
- 地支 = 你凌晨三点喂奶时的样子（没滤镜的）

天干是"看起来怎样"，地支是"实际上怎样"。

---

## 地支的三重身份

每个地支都有三个功能：

### 1️⃣ 代表时间

十二个时辰，每个两小时：

| 地支 | 时间 |
|:----:|:----:|
| 子 | 23-1点 |
| 丑 | 1-3点 |
| 寅 | 3-5点 |
| 卯 | 5-7点 |
| 辰 | 7-9点 |
| 巳 | 9-11点 |
| 午 | 11-13点 |
| 未 | 13-15点 |
| 申 | 15-17点 |
| 酉 | 17-19点 |
| 戌 | 19-21点 |
| 亥 | 21-23点 |

### 2️⃣ 代表月份

寅=正月春天，卯=二月，辰=三月……子=十一月，丑=十二月。

### 3️⃣ 代表生肖

子鼠、丑牛、寅虎、卯兔、辰龙、巳蛇、午马、未羊、申猴、酉鸡、戌狗、亥猪。


> 
> **问：** 那同一个地支的人性格都一样吗？**
>
> 不是。比如都属牛（丑），但生在春天（木旺）的牛和生在冬天（水旺）的牛完全不一样。**地支是静态的，月令（月支）决定这个地支能量强不强。**

## 举个例子

假设一个人的八字地支是：**寅 午 辰 申**

- **寅**（年支，出生年份环境）→ 木旺，属虎。出生那年像"春天开局"，底色有冲劲
- **午**（月支，成长环境）→ 火旺，属马。成长过程里热情洋溢、氛围开明
- **辰**（日支，婚姻/内心）→ 土，属龙。内在包容多变，能扛事也能转弯
- **申**（时支，晚年/孩子）→ 金，属猴。机灵善变，晚年爱琢磨新东西

---

## 本课小结

- 地支是"底色"，决定**原生环境和内在状态**
- 每个地支有**时间、月份、生肖**三重身份
- 天干是"演出来的"，地支是"骨子里的"
- 阴阳五行属性都要看——**火和土不一样，阳水和阴水也不一样**
