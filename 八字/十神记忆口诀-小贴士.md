# 十神不用背？两个步骤搞定

> 椰子要破壳 · 八字入门 · 小贴士

---

十神有10个，名字长得像亲戚——正官七杀、正印偏印、食神伤官、正财偏财、比肩劫财。

刚学谁分得清。**两个步骤就够。**

---

## 第一步：记住5种关系

不管阴阳，先看你（日主）跟别人是什么关系。

以你（木）为中心：

▶ **克你的（金）** = 管你的人
→ 领导、规矩、压力、老公

▶ **生你的（水）** = 帮你的人
→ 贵人、长辈、学习、文凭

▶ **你生的（火）** = 你的才华
→ 说话、写作、作品、表达

▶ **你克的（土）** = 你的钱
→ 工资、收入、财产、你能驾驭的东西

▶ **同你的（木）** = 你的朋友
→ 同事、兄弟姐妹、合作伙伴

**画面记忆：**
- 金克木 = 斧头砍树（管你）
- 水生木 = 浇水养花（帮你）
- 木生火 = 烧柴做饭（你生出来的才华）
- 木克土 = 种地收获（你能赚到的钱）
- 木见木 = 两棵藤长一起（朋友）

---

## 第二步：分阴阳 = 10个十神

每1种关系分阴阳，变成2个十神。

**怎么分？** 看跟你（日主）的阴阳一样不一样。

---

**克你的（金）：**
- 庚金（阳）→ 阴阳相反 → **正官**（温柔管你）
- 辛金（阴）→ 阴阳相同 → **七杀**（严格管你）

**生你的（水）：**
- 壬水（阳）→ 阴阳相反 → **正印**（妈妈型贵人）
- 癸水（阴）→ 阴阳相同 → **偏印**（老师型贵人）

**你生的（火）：**
- 丙火（阳）→ 阴阳相反 → **伤官**（大火烧山，敢说敢写）
- 丁火（阴）→ 阴阳相同 → **食神**（烛光，安静有才）

**你克的（土）：**
- 戊土（阳）→ 阴阳相反 → **正财**（工资稳定收入）
- 己土（阴）→ 阴阳相同 → **偏财**（外快投资收入）

**同你的（木）：**
- 甲木（阳）→ 阴阳相反 → **劫财**（跟你抢的朋友）
- 乙木（阴）→ 阴阳相同 → **比肩**（跟你并肩的朋友）

---

## 规律总结

阴阳相反 → 叫"正"：正官、正印、正财、伤官（例外）、劫财（例外）

阴阳相同 → 叫"偏/七"：七杀、偏印、偏财、食神、比肩

> 注意：伤官和劫财名字里没有"正"字，但它们属于阴阳相反那一类。以表格为准。

---

## 一句话

1. 先定5种关系：克你的 / 生你的 / 你生的 / 你克的 / 同你的
2. 再分阴阳：相反叫"正"，相同叫"偏"
