第4课：五行关系——能量是怎么流动的

椰子要破壳 · 八字入门系列 · 第4课

前面学了天干和地支。不管天干地支，它们背后的"能源"都是五个字——木火土金水。

今天不讲新的，就讲这五个字怎么互相影响。

用画面记五行生克

如果从来没接触过五行，没关系。一个画面就够了——做饭。

生（相生）：A生B，A是B的妈

木生火 → 火生土 → 土生金 → 金生水 → 水生木

在厨房里： 1. 木生火 — 往灶里添柴（木），火烧起来。木头是火的老妈 2. 火生土 — 火烧完，灰掉地上变成土。火是土的老妈 3. 土生金 — 土里挖出矿石，炼成金属。土是金的老妈 4. 金生水 — 用金属工具挖井，水出来。金是水的老妈 5. 水生木 — 浇水，花草树木长起来。水是木的老妈

带娃版： - 宝宝吃奶（水）→ 长大长高（木的生长）→ 活蹦乱跳（火的热能）

克（相克）：A克B，A是B的"管"

木克土 → 土克水 → 水克火 → 火克金 → 金克木

还在厨房： 1. 木克土 — 树根扎进土里，把土撑裂。木头管着土地 2. 土克水 — 土垒堤坝，挡住洪水。泥土管着水 3. 水克火 — 烧水时，水把火温度压住了。水管着火 4. 火克金 — 高温把金属烧化重塑。火管着金属 5. 金克木 — 斧头砍树，锯子锯木头。金属管着木头

带娃版： - 宝宝积食（土太满）→ 运动（木疏土），肚子通了 - 宝宝发烧（火太旺）→ 多喝水（水灭火），体温降了

问： 怎么记住谁克谁？**

五行排一圈：木 - 火 - 土 - 金 - 水。隔一个就是克：

木—土—水—火—金—木

木克土，土克水，水克火，火克金，金克木。

就五句。记住"斧头砍树=金克木"这一个画面，其他类推。

为什么这些关系重要

因为八字里的五行不是孤立的。

比如一个人的八字日主是乙木（草），地支里有酉金，金克木。说明这个人长期受"修剪"： - 酉金的修剪=高标准、环境压力、自我要求高 - 但地支里如果有亥水，水生木，说明有人暗中支持

所以看八字不是看"乙木好不好"，是看： 谁在生你（帮你），谁在克你（压你）。

两个特殊情况

生得太过——"好"过了头

妈妈太爱孩子，什么事都替他做——孩子反而废了。五行里叫"生多为克"： - 水太多 → 木被泡烂 - 火太多 → 土被烤焦 - 土太多 → 金被埋

克得适当——"管"得刚刚好

妈妈适当管教孩子，孩子反而长得好。五行里叫"克以为生"： - 金克木 → 把歪枝剪掉，树才长直 - 水克火 → 别烧太旺，细火慢炖才入味

八字里没有绝对的好和坏，只有"过了"和"刚刚好"。

一个简单的自查方法

拿到八字就看三件事：

  1. 日主是什么五行（比如木、土）
  2. 谁的五行最多（能量最强的是哪个？）
  3. 谁在生、谁在克（帮的人多还是压的人多？）

比如一个戊土日主的小朋友： - 八字里火多 → 火生土，基础体质不错 - 但金少 → 金是肺、免疫力，容易感冒 - 对策：补金（白色食物、润肺）

本课小结

  • 相生：木→火→土→金→水→木（循环生）
  • 相克：木→土→水→火→金→木（循环克）
  • 生是"帮"，克是"管"
  • 过度的生＝害，适当的克＝利
  • 八字不是判刑，是看能量平衡图
